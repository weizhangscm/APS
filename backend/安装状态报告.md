## 推荐解决方案

### 方案 1：手动删除环境变量（最可靠）

1. 按 `Win + R`，输入 `sysdm.cpl`，回车
2. 点击"高级" → "环境变量"
3. 在"用户变量"中找到 `PIP_NO_INDEX`，删除它
4. **重新打开 PowerShell**（重要！）
5. 运行：
   ```powershell
   cd C:\Users\Chun.Wang\Desktop\APS\backend
   python -m pip install --index-url https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com -r requirements.txt
   ```

### 方案 2：检查并配置代理

如果您的网络需要代理：

1. 检查系统代理设置：
   - 设置 → 网络和 Internet → 代理
   - 查看是否配置了代理服务器

2. 如果配置了代理但无法连接，可以：
   - 临时禁用代理
   - 或配置正确的代理地址

3. 然后重新运行安装

### 方案 3：使用离线安装（如果网络持续有问题）

1. 在有网络的机器上下载 wheel 文件：
   ```powershell
   pip download -r requirements.txt -d packages --index-url https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com
   ```

2. 将 packages 文件夹复制到当前机器

3. 离线安装：
   ```powershell
   pip install --no-index --find-links packages -r requirements.txt
   ```
