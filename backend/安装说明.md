# 后端依赖安装说明

## 遇到的问题

安装后端依赖时遇到了以下问题：

1. **PIP_NO_INDEX 环境变量**：系统环境变量中设置了 `PIP_NO_INDEX=1`，这会阻止 pip 从 PyPI 下载包
2. **代理配置问题**：系统配置了代理，但代理服务器无法连接
3. **权限问题**：临时目录权限不足，导致无法解压安装包（错误：Permission denied）
4. **Python 版本路径**：安装脚本中指定的 Python314 路径不存在，实际为 Python313

## 解决方案

### 方法 1：以管理员身份安装（推荐，解决权限问题）

**重要**：如果遇到权限错误（Permission denied），请以管理员身份运行 PowerShell，然后执行：

```powershell
# 进入后端目录
cd C:\Users\Chun.Wang\Desktop\APS\backend

# 清除阻止 pip 的环境变量
Remove-Item Env:PIP_NO_INDEX -ErrorAction SilentlyContinue
Remove-Item Env:HTTP_PROXY -ErrorAction SilentlyContinue
Remove-Item Env:HTTPS_PROXY -ErrorAction SilentlyContinue
Remove-Item Env:http_proxy -ErrorAction SilentlyContinue
Remove-Item Env:https_proxy -ErrorAction SilentlyContinue

# 禁用系统代理
[System.Net.WebRequest]::DefaultWebProxy = $null
$env:PIP_NO_INDEX = ""
$env:NO_PROXY = "*"

# 使用阿里云镜像源安装依赖（如果清华镜像无法访问）
C:\Users\Chun.Wang\AppData\Local\Programs\Python\Python313\python.exe -m pip install --user --index-url https://mirrors.aliyun.com/pypi/simple/ --trusted-host mirrors.aliyun.com -r requirements.txt
```

**或者使用清华镜像源**：
```powershell
C:\Users\Chun.Wang\AppData\Local\Programs\Python\Python313\python.exe -m pip install --user --index-url https://pypi.tuna.tsinghua.edu.cn/simple --trusted-host pypi.tuna.tsinghua.edu.cn -r requirements.txt
```

### 方法 2：使用安装脚本

运行已创建的安装脚本：

```powershell
cd C:\Users\Chun.Wang\Desktop\APS\backend
[System.Net.WebRequest]::DefaultWebProxy = $null
$env:PIP_NO_INDEX = ""
powershell -ExecutionPolicy Bypass -File install_deps.ps1
```

### 方法 3：永久清除系统环境变量

如果需要永久清除这些环境变量，需要：

1. 打开"系统属性" -> "高级" -> "环境变量"
2. 在"用户变量"或"系统变量"中找到以下变量并删除：
   - `PIP_NO_INDEX`
   - `HTTP_PROXY`
   - `HTTPS_PROXY`
   - `http_proxy`
   - `https_proxy`
3. 或者将代理设置为正确的代理服务器地址

### 方法 4：配置正确的代理

如果您的网络需要通过代理访问外网，请配置正确的代理：

```powershell
$env:HTTP_PROXY = "http://your-proxy-server:port"
$env:HTTPS_PROXY = "http://your-proxy-server:port"
```

然后运行安装命令。

## 如果仍然遇到权限问题

如果以管理员身份运行仍然遇到权限问题，可以尝试：

1. **临时禁用防病毒软件**：某些防病毒软件可能会阻止 pip 在临时目录中解压文件
2. **手动清理临时目录**：
   ```powershell
   Get-ChildItem "$env:TEMP\pip-*" -ErrorAction SilentlyContinue | Remove-Item -Recurse -Force -ErrorAction SilentlyContinue
   ```
3. **使用虚拟环境**：创建项目专用的虚拟环境（推荐用于生产环境）
   ```powershell
   python -m venv venv
   .\venv\Scripts\Activate.ps1
   pip install -r requirements.txt
   ```
