# 班次管理功能 - 新增说明

## 功能概述

在"主数据管理"菜单下新增了"班次"管理功能，用于管理资源的工作班次信息。

## 实现内容

### 1. 创建的文件

**前端页面**：`frontend/src/views/ShiftsView.vue`
- 功能完整的班次管理页面
- 包含列表展示、新增、编辑、删除功能

### 2. 修改的文件

#### a) 路由配置 (`frontend/src/router/index.js`)

添加了班次路由：
```javascript
{
  path: '/master-data/shifts',
  name: 'Shifts',
  component: () => import('@/views/ShiftsView.vue'),
  meta: { requiresAuth: true }
}
```

#### b) 菜单配置 (`frontend/src/App.vue`)

在"主数据管理"子菜单中添加了"班次"菜单项：
```vue
<el-menu-item index="/master-data/shifts">班次</el-menu-item>
```

菜单顺序：
1. 工作中心
2. **班次**（新增）
3. 资源
4. 产品
5. 工艺路线
6. 切换矩阵

## 字段说明

班次管理包含以下字段：

| 字段 | 类型 | 说明 | 必填 |
|------|------|------|------|
| 资源 | 下拉选择 | 选择对应的资源 | 是 |
| 班次 | 文本 | 班次代码（如：SHIFT1） | 是 |
| 班次名称 | 文本 | 班次名称（如：早班） | 是 |
| 开始时间 | 时间 | 班次开始时间（格式：HH:mm） | 是 |
| 结束时间 | 时间 | 班次结束时间（格式：HH:mm） | 是 |
| 休息时间 | 数字 | 休息时间（分钟），范围0-480 | 否 |

## 功能特性

### 1. 列表展示
- 表格形式展示所有班次
- 显示资源名称（自动从资源ID映射）
- 显示班次的所有信息

### 2. 新增班次
- 点击"新增班次"按钮打开对话框
- 表单验证确保数据完整性
- 休息时间以15分钟为步进单位

### 3. 编辑班次
- 点击列表中的"编辑"按钮
- 在对话框中修改班次信息

### 4. 删除班次
- 点击"删除"按钮
- 二次确认防止误删

### 需要后续开发的部分

1. **后端API**（需要开发）：
   - `GET /api/shifts` - 获取班次列表
   - `POST /api/shifts` - 创建班次
   - `PUT /api/shifts/{id}` - 更新班次
   - `DELETE /api/shifts/{id}` - 删除班次

2. **数据库表**（需要创建）：
   ```sql
   CREATE TABLE shifts (
       id INTEGER PRIMARY KEY,
       resource_id INTEGER NOT NULL,
       shift_code VARCHAR(50) NOT NULL,
       shift_name VARCHAR(100) NOT NULL,
       start_time VARCHAR(5) NOT NULL,  -- HH:mm格式
       end_time VARCHAR(5) NOT NULL,    -- HH:mm格式
       break_time INTEGER DEFAULT 0,    -- 分钟
       created_at DATETIME,
       updated_at DATETIME,
       FOREIGN KEY (resource_id) REFERENCES resources(id)
   );
   ```

3. **与资源列表的集成**：
   - 需要从实际的资源API获取数据
   - 当前使用的是Mock资源数据
